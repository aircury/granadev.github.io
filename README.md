# Granadev

Granadev is a modern, responsive landing page for a software solutions company based in Granada, Spain. The site showcases services, company information, and contact details, with support for multiple languages and a clean, professional design.

## Features
- Responsive design using Tailwind CSS
- Custom color palette and fonts
- Language switcher (English/Spanish)
- Service and about sections
- Contact information and business hours
- Social media links

## Project Structure

## Getting Started

1. **Clone or download the repository.**
2. Open `index.html` in your web browser to view the site locally.
3. To enable language switching, ensure `lang-switcher.js` is present in the same directory.

## Customization
- Update company information, services, and contact details in `index.html` as needed.
- Replace images (logo, background, etc.) with your own branding.
- Modify colors and fonts in the Tailwind configuration section inside the `<head>` of `index.html`.

## Deployment
You can deploy this site on any static hosting service (e.g., GitHub Pages, Netlify, Vercel) by uploading the project files.
